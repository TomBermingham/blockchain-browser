How it works

Overview

There are two parts: the client and the smart contract.

The smart contract

The smart contract has a function called retrieve() which delivers a string of HTML. 

            
// SPDX-License-Identifier: GPL-3.0

pragma solidity >=0.8.2 <0.9.0;

/**
 * @title Page
 * @dev Store & retrieve value in a variable
 */
contract Page {

    string text = '<article> <header> <h3>Example</h3></header> <p>Here is some content.</p> </article>';

    /**
     * @dev Return value 
     * @return value of 'text'
     */
    function retrieve() public view returns (string memory){
        return text;
    }
}
The client

The root page acts as a hypermedia client (web browser). Its job is to read and display the smart contract's HTML to the user.

This page is brought to you by an Ethereum smart contract.
